Markdown Documentation

Introduction

Markdown is a popular, lightweight and easy to use markup language created by John Gruber in 2004.


It is very similar to HTML (HyperText Markup Language), and is widely used for web content.


This documentation will act as a cheatsheet to demonstrate how you can use Markdown.


The literal way to type out Markdown code will be presented first in the code boxes, and the visual output will be shown below it.


Note: This website was created for my own learning purposes, not meant for actual use.
Instead you can use this comprehensive Markdown guide to learn how to use Markdown.

Headers

To create headers, use the hash/pound sign # followed by a space before text.

Text

Emphasis

To create bold text use, 2 asterisks **, or 2 underscores __ enclosing text.

**Bold text**
Bold text

or

__Bold text__
Bold text

Italic

To create italic text, use 1 asterisk *, or 1 underscore _ enclosing text.

*Italic text*
Italic text

Combined Bold and Italic

To create combined bold and italic text, use 1 underscore _ and 2 asterisks **, or use 3 underscores ___, or use 3 asterisks *** enclosing text.

_**Bold and italic text**_
Bold and italic text

or

***Bold and italic text***
Bold and italic text

or

___Bold and italic text___
Bold and italic text
Lists

Bullet List

To create a bullet list, use a hyphen -, or a plus +, or an asterisk * on a new line before an item in the list.

- List item
+ List item
* List item
  • List item
  • List item
  • List item

Numbered List

To create a numbered list, use numbers starting from 1 followed by a period . on a new line before an item in the list.

1. List item
2. List item
3. List item
  1. List item
  2. List item
  3. List item
Links

To create a link, first enclose the display text in square brackets [] followed by the page url in parenthesis ().

[Link text](Link URL)
Link text
Blockquotes

To create blockquotes, use the greater than symbol > on a new line followed by text.

> Blockquote text

Blockquote text
Tables

To create a table, use the pipe symbol | to create columns, and at at least 3 hyphens - to create lines.

|Header 1 |Header 2|
| ------- | ------ |
|data row |data row |
Header 1 Header 2
data row data row
Code Blocks

To create code blocks, enclose text using backticks `.
